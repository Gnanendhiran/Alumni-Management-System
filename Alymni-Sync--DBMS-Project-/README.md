## 📌 Project Overview
The Alumni Management System bridges the gap between current students and graduated alumni. It features role-based access, a job board, event management, private direct messaging, and searchable directories.

---

## ⚡ Key DBMS Features
* **3NF Normalized Database:** Fully structured relational schema built using the MySQL InnoDB engine.
* **Database Triggers:** Automates counter updates (like comment counts, like counts, and job applications) directly in the database.
* **Analytical Views:** Pre-compiled SQL views (e.g., dashboard statistics and active jobs) for fast, optimized queries.
* **SQL Injection Security:** Uses parameterized queries throughout the Flask backend to block SQL injections.
